Привет всем!
Это программа разработана командой за короткий промежуток времени (3 дня).
Программа преобразовывает картинку из формата jpg, bmp и gif в картинку из символов кодировки UTF-8.
В этой папке есть пробные фотографии, которые можно конвертировать в ASCII-art.

Для начала нужно запустить исходный код в MinIDE, который можно скачать из сайта:
https://ipc.susu.ru/46600.7z

Далее необходимо перейти во вкладку "Настройки", далее "Открыть файл для локальных настроек" и, необходимо, прописать "build.goal=all"

Далее необходимо перейти во вкладку "Действия", далее "Собрать проект -> F7", далее "Выполнить ->F5"

Если при "F7" появляется ошибка, то попробуйте во вкладке "Действия", далее "Очистить от результатов сборки -> Shift + F7".

Hello everybody!
This program was developed by the team in a short period of time (3 days).
The program converts an image from jpg, bmp and gif format into an image from UTF-8 characters.
There are trial photos in this  folder that can be converted to ASCII-art.

First, you need to run the source code in MinIDE, which can be downloaded from the site:
https://ipc.susu.ru/46600.7z

Next, go to the "Settings" tab, then "Open the file for local settings" and, if necessary, register "build.goal=all"

Next, go to the "Actions" tab, then "Build a project -> F7", then "Execute ->F5"

If an error appears at "F7", then try in the "Actions" tab, then "Clear from the build results -> Shift + F7".
